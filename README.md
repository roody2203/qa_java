# Sprint_6

This project was completed as part of the 6th Yandex Practicum sprint.
The project includes automated Unit tests in Java using JUnit.

### Run the project

- Go to the project folder
- Launch the project
```
mvn clean test
```

### Code coverage

Jacoco code coverage repors is located in 
```
target/site/jacoco/.
```

### Requirements
| Dependency | Version |
| ------------ | ------------ |
| JDK | 11.0.26  |
| JUnit | 4.13.2 |
| Maven | 11 |
| Mockito | 5.17.0 |
| ByteBuddy | 1.17.5 |
| Jacoco | 0.8.13 |
